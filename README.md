## 🌟 Можливості

- ✅ **Повністю безкоштовно** — жодних підписок чи оплат
- ✅ **Без лімітів** — трафік не обмежений
- ✅ **No logs** — не зберігаємо жодних логів
- ✅ **Обфускація трафіку** — працює в регіонах з DPI (Росія, Китай, Іран тощо)
- ✅ **Kill Switch** — блокує інтернет при розриві VPN
- ✅ **Автоматичний вибір сервера** — найшвидший пінг
- ✅ **Мульти-протокол** — SOCKS5, HTTP, TLS обфускація
- ✅ **Системний трей** — працює в фоновому режимі
- ✅ **CMD команди** — керування через командний рядок
- ✅ **Автозапуск** — запускається разом з Windows
- ✅ **Встановлення в систему** — як звичайна програма

## 🚀 Швидке встановлення

### Спосіб 1: Інсталятор (рекомендовано)

1. Завантажте [`PoowoVPN_Setup.exe`](https://github.com/poowo1101/PoowoVPN/releases/latest)
2. Запустіть інсталятор
3. Дотримуйтесь інструкцій
4. Готово! PoowoVPN встановлено

### Спосіб 2: Портативна версія

1. Завантажте [`PoowoVPN.exe`](https://github.com/poowo1101/PoowoVPN/releases/latest)
2. Запустіть — не потребує встановлення

## 💻 Команди CMD

Після встановлення доступні команди в командному рядку:

```cmd
poowo                    # Запуск PoowoVPN
poowo --connect          # Підключити VPN
poowo --disconnect       # Відключити VPN
poowo --status           # Статус підключення
poowo --servers          # Список доступних серверів
poowo --speedtest        # Тест швидкості
poowo --install          # Встановити в систему
poowo --uninstall        # Видалити з системи
poowo --help             # Довідка
